Langchain: Open Source framework that helps in building LLM based applications. It provides modular components and end-to-end tools that help developers build complex AI applications, such as chatbots, question-answering systems, retrieval-augmented generation(RAG), autonomous agents, and more. 

1. Support all the major LLMs.
2. Simplifies developing LLM based applications. - (Concepts such as chain where we can create complex pipelines.)
3. Integrations available for all major tools.
4. Opensource/Free/Actively developed.
5. Supports all major GenAI use cases.
